# 学术知识库项目

## 项目结构
```
project/
├── backend/         # Python后端
│   ├── app.py       # FastAPI主应用
│   ├── models.py    # 数据模型
│   ├── routes/      # API路由
│   └── requirements.txt
├── frontend/        # 前端资源
│   ├── public/      # 静态资源
│   ├── src/         # 源代码
│   └── package.json
└── docs/            # 项目文档
```

## 技术栈
- 后端：Python + FastAPI
- 前端：Vue.js + TailwindCSS
- 数据库：SQLite (开发环境)
